Why this layout

One control axis, shown two ways.

上层是交互式对比，下层是同图并排全量展示。这样不会丢掉滑块的沉浸感，也不会牺牲对强度递进的整体理解。

Scrollable showcase

Every card keeps one image together.

每张卡片都先用滑块选当前强度，再把同一张图的 `Input / 0.25 / 0.50 / 0.75 / 1.00` 并排摆在一起，对比会更明显。
